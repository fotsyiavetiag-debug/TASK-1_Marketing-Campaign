# Marketing Campaign Analysis

## Project Overview

This project analyzes customer data from a marketing campaign dataset to identify customer behavior patterns and provide actionable business insights.

The analysis includes:
- Data cleaning
- Missing value treatment
- Exploratory Data Analysis (EDA)
- Customer segmentation insights
- Marketing performance evaluation

---

## Dataset

Dataset: Marketing Campaign

Main variables include:

- Customer ID
- Year of Birth
- Education
- Marital Status
- Income
- Product Spending
- Campaign Responses
- Purchase Channels

---

## Project Objectives

The main goals are:

1. Clean and prepare the dataset.
2. Handle missing values.
3. Explore customer demographics.
4. Analyze spending behavior.
5. Identify characteristics of high-value customers.
6. Generate recommendations for future marketing campaigns.

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Data Cleaning

Performed the following cleaning steps:

- Converted column names to lowercase
- Removed inconsistencies
- Handled missing values
- Checked duplicates
- Validated data types
